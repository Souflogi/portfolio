# Soufiane Sammah — Portfolio

Single‑page portfolio built with React, Vite, React Router, and CSS Modules. It features a sticky sidebar navigation, responsive layout, and simple project/contact sections.

## Scripts

- `npm install` — install dependencies
- `npm run dev` — start local dev server
- `npm run build` — production build
- `npm run preview` — preview production build

## Tech

- React 18, Vite 6
- React Router (DOM)
- CSS Modules, custom CSS variables
- React Icons

## Structure

- `src/pages/home` — hero and social links
- `src/pages/about` — story + skills grid
- `src/pages/works` — projects grid
- `src/pages/contact` — contact + CTA
- `src/components` — NavBar, Logo, Button, SkillItem

## Notes

- Update the projects data in `src/pages/works/WorkPage.jsx` with real demo/code links as they are ready.
- Email is configured in `src/pages/contact/ContactPage.jsx`.
